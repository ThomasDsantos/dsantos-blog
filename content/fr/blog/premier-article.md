# Bienvenue sur mon blog

Je suis ravi de vous présenter mon nouveau blog et portfolio, créé avec les dernières technologies web.

## Technologies utilisées

Ce site est construit avec :

- **Nuxt 4** - Le framework Vue.js pour la production
- **Nuxt UI** - Une bibliothèque de composants moderne et élégante
- **Nuxt Content** - Pour gérer facilement le contenu en Markdown
- **Tailwind CSS** - Pour un design responsive et moderne

## Pourquoi ce blog ?

J'ai créé ce blog pour partager mes connaissances et mes expériences dans le développement web. Vous trouverez ici :

- Des tutoriels techniques
- Des réflexions sur les technologies modernes
- Des retours d'expérience sur mes projets
- Des conseils et bonnes pratiques
